- Ajuste de styles da pagina html para o css:

h1{
    text-align: center;
}
--------------------------------------------------------------------------------------------------------------------------------------------------------------------
- marcando um paragrafo especifico no CSS:
 HTML: <p id="missao"><em> Nossa missão é: <strong> "Proporcionar auto-estima e qualidade de vida aos clientes"</strong>.</em></p>
 CSS: 
 
 p {
    text-align: center;
}

#missao {
    font-size: 20px
}
--------------------------------------------------------------------------------------------------------------------------------------------------------------------
- Adicionar uma imagem na pagina:
<img src="banner.jpg">

- Syle para uma imagem:

HTML: <img id="banner"  src="banner.jpg">
CSS: 
#banner{
   width: 100% 
}
